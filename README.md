Multifunctional toolbox
这个项目是我用PyQt自主设计开发的多功能工具箱项目，这个项目不仅仅有网络助手、蓝牙助手、串口助手、还有串口上位机，上位机有很多功能，通过连接下位机就可以进行操作，我所连接的下位机是我开发的蓝牙寻线小车项目，如果想要操作别的项目都可以进行修改实现。
